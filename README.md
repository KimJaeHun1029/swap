## Prerequisites

- Node.js 18+
- pnpm (the repo uses pnpm workspaces)

## Accessing the KyberSwap Interface

To access the KyberSwap Interface, visit [kyberswap.com](https://kyberswap.com/)

## Development

### Install Dependencies

```bash
pnpm i
```

### Build packages

```bash
pnpm build-package
```

### Run

```bash
cd apps/kyberswap-interface && pnpm start
```
